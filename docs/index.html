Method, sources & caveats

  • Distances are straight-line from the HDB building centroid — an easy, consistent yardstick, but real walks are longer (roads, blocks, canals). MRT/LRT distance is to the nearest station exit, not the platform.
  • Default benchmarks (adjustable above): supermarket / hawker / clinic 500 m, MRT 800 m, park 800 m, school / mall / community club 1 km, library 2 km. These are editorial yardsticks, not official standards.
  • HDB blocks: HDB Existing Building footprints (data.gov.sg) — includes every HDB building, so some points are multi-storey carparks or commercial blocks.
  • Supermarkets: SFA licensed-supermarket register (data.gov.sg), filtered to NTUC FairPrice and Sheng Siong licences, geocoded by postal code via OneMap. Licence address ≈ store address. Other chains (Giant, Cold Storage, Prime…) are excluded by design.
  • Primary schools: MOE School Directory (data.gov.sg), PRIMARY and mixed-level (P1–S4) schools, geocoded by postal code. Note: home-to-school priority uses 1 km/2 km as defined by SLA for the address, which differs slightly from centroid distance.
  • Malls: editorial list of ~110 major malls (Wikipedia), geocoded via OneMap search — boutique/strip malls omitted.
  • MRT/LRT: LTA station-exit layer (data.gov.sg), including LRT; station markers are the mean of their exits.
  • Hawker centres: NEA register (data.gov.sg). Clinics: CHAS GP clinics, MOH (dental excluded). Community clubs: PA. Libraries: NLB. Parks: NParks managed-area points.
